Move Sidepanel to Left

BarCat works best when the sidepanel is positioned on the left side of your browser.

How to change sidepanel position:

  1. Go to Chrome Settings (‚ãÆ menu > Settings)
  2. Click on "Appearance" in the sidebar
  3. Scroll down to "Side panel position"
  4. Select "Show on left"

Note: This option might not be available in all Chromium-based browsers like Edge and Comet.

Set Up Keyboard Shortcuts

Configure keyboard shortcuts to quickly access BarCat features from anywhere in your browser.

How to set up keyboard shortcuts:

  1. Open Chrome Extensions page (chrome://extensions/shortcuts)
  2. Go to the keyboard shortcuts section
  3. Find BarCat and customize the shortcuts
